Welcome to my media tracking website!

What can you do on this website?

Currently, the user is able to search for movies and games on the Movies and Games page. After searching for a movie or game, the user is able to add it to a list. The list will contain a poster for the movie or game, the title, and a general overview of the media. The user is also able to remove it from their list if needed. The website is very barebones at the moment. For more information about me, the developer, and information about possible future features, please check the About page.

Where to find my project requirements:

  • This is the home page. The three subpages are the Movies, Games, and the About page.
  • The carousel is at the top of the homepage.
  • A GET request will be sent to TheMoviesDB or RAWG after utilizing the search form on the Movies or Games page.
  • The results of the search will be returned in a scroll box at the top of the Movies or Games page.
  • The search results are stored as an unordered list. This list is also an unordered list.
  • A Metacritic link will appear to take the user to an outside webpage after adding a game to their list.
  • There is a download button at the bottom of the About page to download my code.
